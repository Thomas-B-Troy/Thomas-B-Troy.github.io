## About

A single self-contained `index.html` — no build step, no dependencies. Open it in any browser and it runs. Hosted free on GitHub Pages.

## Contents

- `index.html` — the complete website (styles, scripts, and images bundled inline)

## Updating the site

The site is edited in a separate design file and re-exported as a single `index.html`. To publish an update, replace `index.html` in this repo (**Add file → Upload files**) and commit — GitHub Pages redeploys automatically within a minute.
